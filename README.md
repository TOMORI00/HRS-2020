# HRS（Hotel Reservation System）-2020-SEⅡ

NJUSE软件工程课程与计算Ⅱ课程项目

成员DQJ、CSH、LJL、XYX.

运用软件工程过程的相关知识开发一个互联网酒店预订系统,进行软件工程过程的实践与体验

---

说明：

   - 项目仅为课程学习项目,未进行实际部署
   
   - 涉及到的框架与技术：
   
      - Vue：Javascript框架

      - axios：基于promise的HTTP库
      
      - npm：Node.JS包管理工具
      
      - Node.JS：服务端的JavaScript运行环境
      
      - SpringBoot：简化升级的Spring框架

      - Tomcat：Web应用服务器
      
      - Mybatis：java持久层框架

      - Junit：代码测试技术
